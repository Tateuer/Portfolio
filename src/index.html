VerdeVolver

Recycling Website: A cutting-edge website designed to promote recycling initiatives and facilitate environmentally conscious actions.

Key Features:

  • Payment Gateway Integration: Seamlessly integrates with MercadoPago for secure online payments, allowing users to support recycling initiatives.
  • Contact Zone: Includes a dedicated contact section where users can reach out for inquiries, feedback, or collaboration opportunities.
  • Advanced Backend Technologies: Utilizes Express.js and Node.js to create a robust backend infrastructure, with PostgreSQL and Sequelize as the ORM for efficient data management.
  • User Authentication and Authorization: Implements Passport.js and Bcrypt for secure user authentication and authorization, ensuring data privacy and security.

Technologies Used:

  • React: Used for building the frontend components and ensuring a dynamic user experience.
  • Redux: Implemented for state management, enabling efficient data flow and synchronization across components.
  • Nodemailer: Utilized for sending automatic email notifications to users for various events and interactions.
  • Chakra UI: Used for frontend styling to ensure a visually appealing and user-friendly design.
  • Google Maps: Integrated to provide an interactive map experience for locating recycling centers and facilities.

Additional Features & Tools:
